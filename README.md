# 🚀 CareerLift-AI

CareerLift-AI is an AI-powered resume analysis platform that helps students improve their resumes, evaluate ATS compatibility, and receive AI-generated suggestions to enhance their placement readiness.

## ✨ Features

* 🔐 Secure User Authentication (JWT)
* 👤 User Registration & Login
* 📄 Resume Upload & Management
* 🤖 AI-Powered Resume Analysis using Google Gemini
* 📊 ATS Score Evaluation
* 💡 AI-Generated Resume Improvement Suggestions
* 📈 Interactive Dashboard with Resume Statistics
* 👤 User Profile Management
* 📱 Fully Responsive User Interface
* 🐳 Dockerized for Local Development

## 🛠️ Tech Stack
